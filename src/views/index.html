HTML? WHAT?!

Don't worry, we're still going to be using React for everything front-end. This is just to show that you can, in fact, render HTML directly to the browser with Express if you so choose.

Open up that inspect tool! Hover over the image element in the Elements window of the Dev Console. The src attribute shows "./images/KALogo.png", but hovering shows "http://localhost:3001/images/KALogo.png".

This is made possible by line 12 in the the server file. It makes all files in the public folder accessible by simply appending the file's path to the end of http://localhost:3001/code>.
